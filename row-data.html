RAW IMAGES

raw images taken by NASA's

Curiosity Mars Rover

NASA has begun using a new system to process the raw data generated by its Curiosity rover on Mars. The Mission Data Processing and Control System (MPCS) presents data visually so that it's useful to a wider range of project team members, including mission managers, software developers, and scientists. For the past week, Curiosity has been parked in the same spot on Mars as NASA tests the rover's robotic arm in preparation for using its instruments to "touch" rocks for the first time. The space agency plans to resume driving the rover within the next few days, generating new kinds of data, such as the composition of rocks, to be transmitted back to mission headquarters at NASA's Jet Propulsion Laboratory in California.
